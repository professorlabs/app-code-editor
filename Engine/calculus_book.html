Derivatives and Gradient

Partial Derivatives Link

For $f(x,y) = x^2y + 3y$, \[ \frac{\partial f}{\partial x} = 2xy, \quad \frac{\partial f}{\partial y} = x^2 + 3 \]

Vector Form Link

\[ \nabla f(x,y) = \begin{bmatrix} 2xy\\ x^2 + 3 \end{bmatrix} \]

Machine Learning Relevance Link

This is used in gradient-based optimization. % ==== CHAPTER 4 ====

Integration and Probability

Continuous Probability Link

If $f(x)$ is a PDF, then: \[ \int_{-\infty}^{\infty} f(x) dx = 1 \]
